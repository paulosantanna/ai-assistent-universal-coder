# CONTINUOUS_LEARNING.md

Track separately:

- evidence and source changes;
- dataset changes;
- retrieval and index changes;
- model and adapter changes;
- calibration and subgroup metrics;
- simulation assumptions;
- security posture;
- architecture decisions;
- documentation drift.

Use representative historical replay cases to reduce catastrophic forgetting. Candidate models, indexes, prompts, policies, and knowledge remain in shadow mode until independently validated and approved.
