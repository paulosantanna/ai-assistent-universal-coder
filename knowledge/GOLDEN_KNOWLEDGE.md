## Purpose

Define high-confidence knowledge used as default operating truth.

## Promotion criteria

Knowledge becomes golden only when:

- supported by strong evidence;
- repeated across contexts;
- reviewed by judge or human;
- not contradicted by current project evidence;
- has clear applicability boundaries.

## Golden knowledge fields

- statement;
- scope;
- evidence;
- exceptions;
- expiration/review trigger;
- owner;
- version.

## Demotion

Golden knowledge must be demoted when:
- evidence changes;
- project context invalidates it;
- safer practice emerges;
- repeated failures occur.

## Rule

Golden knowledge is strong default guidance, not blind dogma.
