# Fontes técnicas primárias

Estas fontes orientam os contratos do pack. Elas não substituem a inspeção do
repositório alvo.

- Playwright — configuração, projetos, retries, paralelismo, traces e artefatos:
  https://playwright.dev/docs/test-configuration
- FastAPI — metadados, tags e documentação OpenAPI:
  https://fastapi.tiangolo.com/tutorial/metadata/
- RFC 9457 — Problem Details for HTTP APIs:
  https://www.rfc-editor.org/rfc/rfc9457.html
- Pydantic Settings — settings tipados, fontes e secrets:
  https://pydantic.dev/docs/validation/latest/concepts/pydantic_settings/
- OpenTelemetry — traces, metrics, logs e baggage:
  https://opentelemetry.io/docs/concepts/signals/
- Kubernetes — liveness, readiness e startup probes:
  https://kubernetes.io/docs/tasks/configure-pod-container/configure-liveness-readiness-startup-probes/
- C4 Model — Context, Container, Component e Code:
  https://c4model.com/
- pre-commit — execução de hooks versionados:
  https://pre-commit.com/
- release-please — release PRs baseados em commits convencionais:
  https://github.com/googleapis/release-please-action
