# AEOS Core

Kernel runtime and foundational services.

## Modules

| Module | Description |
|--------|-------------|
| Kernel | Central orchestrator, tool router, context manager |
| Context | Sliding window context management |
| Policy | Policy enforcement and rule evaluation |
| Permissions | Access control and permission matrix |
| Tools | Tool definitions and handler registry |
| Memory | Memory bridge to long-term storage |
| Scheduler | Task queue, concurrency, retry |
| Evidence | Evidence collection and validation |
| Judge | Evaluation engine and scoring |
| Audit | Immutable logging and audit trail |
| Rollback | State snapshots and rollback execution |
