# AEOS Major Evolution Pack v0.8 → v1.0

Este pacote transforma o AEOS de um runtime governado básico para um workbench mais completo.

## Macrocamadas

```text
v0.8
  Parallel Execution
  Task Dependency Graph
  Read/Write Set Detection
  Conflict Resolver
  Deterministic Scheduler

v0.9
  Approval Gateway avançado
  Evidence Cache
  Observability
  Notification hooks locais
  Test Runner parsing

v0.95
  Security Hardening
  Pack Marketplace local
  Trust Policy
  Quarantine
  Compliance docs
  Data retention

v1.0
  Workbench Readiness
  Stable CLI surface
  Quality gates
  Release checklist
  Operational runbooks
```

## Princípio

AEOS deve ficar mais poderoso, mas sem perder governança.

```text
Mais poder exige mais controle.
Mais automação exige mais evidência.
Mais agentes exigem mais auditoria.
Mais integração exige mais política.
```
