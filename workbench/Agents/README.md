# AEOS Agents

Agent specifications by role.

## Roles

| Agent | Description |
|-------|-------------|
| Root | Orchestrator, receives and decomposes tasks |
| Architect | Architecture decisions, patterns, tech debt |
| Coder | Code implementation and refactoring |
| Tester | Test creation and execution |
| Security | Security audit and vulnerability detection |
| DevOps | Infrastructure, CI/CD, deployment |
| Documenter | Documentation generation |
| Incident | Incident response and root cause analysis |
| Researcher | Technical research and spike solutions |
| Scraper | Web data collection |
| Judge | Evaluation and quality assurance |
