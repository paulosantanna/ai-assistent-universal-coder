# AEOS Observability

Monitoring, logging, and telemetry.

## Modules

| Module | Description |
|--------|-------------|
| Logs | Structured JSON logging |
| Traces | Distributed tracing |
| Metrics | Token usage, tool calls, execution time |
| TokenUsage | Per-agent and per-task token tracking |
| ToolCalls | Tool call logging with parameters and results |
| ExecutionReports | Auto-generated execution reports |
