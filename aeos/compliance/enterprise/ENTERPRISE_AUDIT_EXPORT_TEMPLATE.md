# ENTERPRISE_AUDIT_EXPORT_TEMPLATE

## Purpose

Enterprise audit export template for regulated environments.

## Required Sections

- Scope
- Systems affected
- Evidence refs
- Risk classification
- Controls applied
- Approvals
- Rollback
- Retention
- Redaction
- Judge decision

## Evidence Required

- policy decisions
- permission decisions
- tool calls
- MCP calls
- evidence manifest
- hash chain
- Judge report
