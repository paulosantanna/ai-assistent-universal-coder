# CHANGE_ADVISORY_BOARD_TEMPLATE

## Purpose

CAB-ready template for controlled changes.

## Required Sections

- Scope
- Systems affected
- Evidence refs
- Risk classification
- Controls applied
- Approvals
- Rollback
- Retention
- Redaction
- Judge decision

## Evidence Required

- policy decisions
- permission decisions
- tool calls
- MCP calls
- evidence manifest
- hash chain
- Judge report
