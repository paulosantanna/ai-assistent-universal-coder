# How to Add an AEOS Playbook

1. Create `aeos/playbooks/<id>.playbook.md`.
2. Follow `AEOS_PLAYBOOK_CONTRACT.md`.
3. Register it in `aeos/registries/playbooks.registry.yaml`.
4. List required agents, skills, MCPs, and LCPs.
5. Define steps and blocking conditions.
6. Define required evidence and rollback.
