# How to Add an AEOS Skill

1. Create a file in `aeos/skills/core` or `aeos/skills/generated`.
2. Follow `AEOS_SKILL_CONTRACT.md`.
3. Register it in `aeos/registries/skills.registry.yaml`.
4. Declare capabilities.
5. Declare forbidden actions.
6. Declare required evidence.
7. Ensure Judge can validate outputs.
