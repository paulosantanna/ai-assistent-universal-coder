# Observability Guide

AEOS should make every execution understandable.

## Minimum reports

```text
execution.log.jsonl
trace.jsonl
metrics.json
cost.json
timeline.md
observability-report.md
```

## Questions AEOS must answer

- What ran?
- Who/which agent ran it?
- Which tools were called?
- Which permissions were evaluated?
- Which policies blocked?
- Which evidence was generated?
- What failed?
- What was the Judge decision?
