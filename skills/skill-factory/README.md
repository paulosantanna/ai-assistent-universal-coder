# AEOS Skill Factory

Canonical AEOS skill generator for creating, validating, packaging and registering immediately consumable skills.

## Use

```bash
python skills/skill-factory/scripts/skill_factory.py create --prompt "create a documentation skill"
```

Generated skills include `SKILL.md`, `README.md`, `AGENT.md`, knowledge layers, memory layers, an honest evaluator, schema, templates, tests and a manifest.
