## Purpose

Govern secure execution.

## Policies

- least privilege;
- no secret exposure;
- no destructive action without approval;
- no external data transfer without permission;
- prompt injection detection;
- audit logs;
- access review.

## Security incident

If a security incident is detected:

1. Stop unsafe work.
2. Preserve evidence.
3. Report scope.
4. Recommend containment.
5. Do not hide or minimize.
